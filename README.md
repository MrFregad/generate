# HF Image Gen — быстрый деплой на Vercel

## Что это
Один эндпоинт `/api/generate?prompt=твой%20промпт`, который дёргает
Hugging Face Inference API (модель FLUX.1-schnell, бесплатная) и
отдаёт готовую картинку. Твой HF-токен хранится только на сервере
(в переменных окружения Vercel) и никогда не попадает в чат или в URL.

## Шаг 1. Получи бесплатный токен Hugging Face
1. Зарегистрируйся на https://huggingface.co
2. Settings → Access Tokens → New token
3. Права: "Read" достаточно
4. Скопируй токен (начинается на `hf_...`)

## Шаг 2. Задеплой на Vercel
Вариант А — через сайт (без терминала):
1. Зайди на https://vercel.com, залогинься (у тебя, похоже, уже есть аккаунт — там же autoshop-market)
2. New Project → Import → загрузи эту папку (можно как zip, Vercel предложит "Deploy" -> Upload)
   Либо: залей папку в свой GitHub-репозиторий и импортни репозиторий в Vercel — этот способ надёжнее для будущих обновлений
3. В настройках проекта → Environment Variables → добавь:
   - Name: `HF_TOKEN`
   - Value: твой токен `hf_...`
4. Deploy

Вариант Б — через терминал (если удобнее):
```bash
npm i -g vercel
cd hf-image-gen
vercel login
vercel --prod
# когда спросит про env переменные, добавь HF_TOKEN, либо после деплоя:
vercel env add HF_TOKEN
```

## Шаг 3. Проверь
После деплоя Vercel даст тебе домен, например:
`https://hf-image-gen-xxxx.vercel.app`

Проверь в браузере:
```
https://hf-image-gen-xxxx.vercel.app/api/generate?prompt=a%20cat%20in%20a%20hat
```
Должна открыться картинка.

## Возможные ограничения
- Первый запрос к "холодной" модели на HF иногда может занять
  20-30 секунд (модель "просыпается") — в редких случаях вернётся
  ошибка вроде "model is loading", тогда просто повтори запрос
  через несколько секунд.
- Бесплатный тариф HF Inference API имеет лимиты на количество
  запросов в час — для личного использования должно хватать с запасом.
